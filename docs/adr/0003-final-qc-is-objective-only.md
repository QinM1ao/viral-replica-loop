# Final QC Is Objective Only

Final video QC blocks only objective hard failures: missing or unreadable output, missing streams, wrong duration, freeze or blank frames, duplicated or missing speech, broken seams, or obvious wrong person, product, wardrobe, mud, or scene. If final technical QC passes, the loop delivers the video and the user judges subjective effect after delivery. Because Seedance is expensive, an objective final failure can trigger at most one targeted Seedance retry; a repeated failure or second paid retry stops with evidence.
