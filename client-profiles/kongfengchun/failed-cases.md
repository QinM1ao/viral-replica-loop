# Failed Cases To Block

这些不是“风格偏好”，是以后自动打回的失败类型。

## 双男模污染

失败表现：

- 画面里出现两个男模。
- 人物像两个人拼出来。
- 分镜使用了多男模参考 sheet。

原因：

- 身份参考混入多个男模图。
- 用了 `male_identity_ref_sheet.jpg`、`male_model_sheet.jpg` 这类 sheet。

以后处理：

- 一个任务只允许一个男模身份锚点。
- 图像生成引用清单里必须能看出只有一个人物身份图。
- 如果出现双男模，直接废稿，不进入 Seedance。

## 陌生包装盒 / 白罐替身

失败表现：

- 产品旁边出现不存在的包装盒、纸盒、礼盒、说明盒。
- 罐子变成普通白罐，缺少孔凤春绿色标识。
- 产品标签变成乱码或其他品牌感。

原因：

- 参考图给得太宽或太杂。
- 提示词没有锁产品正面和开盖白泥。

以后处理：

- 产品正面和开盖白泥分开引用。
- 优先使用紧裁参考图。
- 出现陌生包装直接废稿。

## 灰泥污染

失败表现：

- 脸上或罐内泥膜变灰、黑、绿、火山泥质感。
- 泥膜变稀，像水膜、乳液、泡沫。
- Part1 是白厚泥，Part2 变成灰稀泥。

原因：

- 原片旧产品是灰泥，被模型继承。
- 分镜图阶段没有先把污染挡住。

以后处理：

- 源片灰泥只允许传递镜头动作和节奏，不允许传递材质。
- Shot06 / Shot10 近景必须额外查厚白泥。
- 分镜图没过，不能进入 Seedance。

## 黄泥 / 米色泥污染

失败表现：

- 脸上泥膜或罐内泥膜偏黄、偏米、偏肤色、暖奶油色。
- 分镜里的泥膜比产品开盖参考明显更黄，不是白色厚泥。
- Part1 / Part2 一个偏白、另一个偏黄，视频前后质感不一致。

原因：

- 室内暖光、肤色或木桌颜色污染了泥膜颜色。
- 图像生成只修了包装，没有把白泥材质修到位。

以后处理：

- 黄色/米色泥和灰泥一样是硬失败。
- 回图像快修阶段修，直到上脸泥和开盖泥都接近产品开盖参考的白色厚泥。
- 不能把黄泥分镜交给 Seedance 继续赌。

## 白泥太薄 / 乳液感

失败表现：

- 脸上的白泥只是薄薄一层，像乳液、粉底、透明霜或平刷白色。
- 罐内泥虽然偏白，但没有厚膏体、堆叠、小峰、纹理或边缘。
- 机器白度、冷白阈值 PASS，但肉眼不像清洁泥膜厚泥。

以后处理：

- 颜色 QC 不能替代厚度检查。
- 必须回 image batch 做 `mud_thickness` 快修。
- 提示词明确“raised opaque thick clay paste, ridges, edges, peaks, piled texture”，直到脸上和罐内都能看出厚泥膏体感。

## 洗后参考裁切错误

失败表现：

- 从最终分镜里裁一个洗后脸部面板当 Seedance 参考。
- 裁切图带着分镜光线、压缩、构图残留，不能稳定控制洗后肤感。

以后处理：

- 当前默认不使用独立洗后参考图，以对应 Shot 文案控制结果。
- 只有 profile 显式要求时，才用 approved 男模身份图单独生成洗后脸部特写；生成图通过身份和肤色检查后再作为洗后素材。

## 洗前被洗后亮肤污染

失败表现：

- 开场、涂抹中、清洗前的脸已经非常白净、透亮、细腻。
- 洗前脸和洗后脸差别不明显，前后对比不成立。
- Part1 或 Part2 前半段看起来已经套用了 `afterwash_face` 的白净效果。

以后处理：

- 洗前脸保留轻微油光、毛孔、闭口、暗沉、粗糙或正常未清洁肤感。
- 洗后亮肤参考只能在洗脸/擦净之后使用。
- 出现洗前过白，直接回 image batch 修 `skin_progression`，不进入 Seedance。

## 网页端素材分散

失败表现：

- Seedance 网页端要用的图、音频、提示词、manifest 分散在多个中间目录。
- 用户需要自己翻找哪些是最终版。

以后处理：

- 在停止前建立一个最终输出区，按 Part 分好上传素材，并放完整提示词和说明。
- 中间废稿只留在过程目录，不进入最终输出区。

## 错误涂抹工具

失败表现：

- 用管状挤出。
- 用棒状/刷头涂抹。
- 在手臂上试色。

原因：

- 原片动作或旧美妆产品动作污染。

以后处理：

- 全部改成手指从罐里取泥、指腹上脸。
- 一旦出现管、棒、刷头、手臂试色，直接废稿。

## 场景和服装污染

失败表现：

- 原片不是浴室，却生成浴室。
- 居家镜头服装变成拉链上衣、立领、运动外套。
- 把源视频人物衣服保留下来，只替换脸，导致人物没有完整替换成模特。
- 健身房服装污染到居家镜头。
- Part1 和 Part2 明显像不同衣服、不同衣领/袖型、不同外套或不同场景，但后续要拼成同一条连续视频。

以后处理：

- 先用画面分析确认每个 Part 的真实场景。
- 默认完整换人：服装跟模特图走；身份参考图锁脸、发型、身份和衣服。只有用户明确要求只换脸或保留源衣服时，才允许源视频服装保留。
- 提示词不要写和已通过参考图无关的“浴室/拉链”等词，避免反向污染。
- 多 Part 图必须生成并检查 side-by-side 连续性图；衣服/场景明显割裂时直接回 image batch 修，不进入 Seedance。

## 节奏错位

失败表现：

- “孔凤春清洁泥膜”没有说在产品特写上。
- 只跟 ASR，不跟字幕和画面，导致镜头点位漂。
- 后半段说话很赶，像强行塞词。

原因：

- 把 ASR 当唯一时间线。
- 没有重建“字幕 + ASR + 画面”逐镜头表。

以后处理：

- 字幕存在时，字幕是节奏主锚。
- ASR 用来补充听感、停顿、无字幕口播。
- 每段 Seedance 提示词前必须有逐镜头表。

## 接缝冻结

失败表现：

- 第二段开头人物头像静止停住。
- Part1 / Part2 接缝像卡了一帧。

原因：

- 接缝只接画面，没有设计起始运动。

以后处理：

- Part2 第一秒必须写清“已在运动中开始”。
- 接缝可以连续，但不能静止。
- 口播边界要留短暂停顿，动作不能停。

## 洗后效果被“真实感”削弱

失败表现：

- 洗后只写“油光减少”“更自然”。
- 提示词追加“保留真实毛孔和皮肤纹理”，导致黑头、闭口、粉刺、油光或粗大毛孔继续明显存在。

以后处理：

- 保持同一人物身份，但洗后结果按当前 profile 正向写满：干净透亮、平滑细腻，问题肌特征消失，近距离完全无瑕疵。
- 默认不用独立洗后参考图；先修对应 Shot 的结果文案。

## B-roll 与主播收口混块

失败表现：

- 同一个执行块同时包含主播同期口播、静物 B-roll 和画外旁白。
- B-roll 后切回人物参考图背景，或重新从中景起手把产品推近，造成尾部硬切生硬。

以后处理：

- 声音模式、场景或画面功能变化时必须新开执行块；静物 B-roll 单独成块。
- B-roll 前一块结束时产品已到目标近景；后一块开镜直接回原片同一场景和相近产品景别，保持硬切，不增加溶解。
