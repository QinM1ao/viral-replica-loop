# 口播文案模板

## 结构要求

先读取 `voiceover-capacity-and-compression.md`。口播长度按每个 Part 实际映射的源片字速自适应计算，不再使用固定的“30秒150-200字”。

```
[前3秒 - 钩子] → 对应 Shot 01-02
[4-15秒 - 痛点/场景] → 对应 Part1 剩余镜头
[16-30秒 - 解决方案/行动号召] → 对应 Part2 全部镜头
```

## 钩子模板（前3秒）

### 问题型
- "你是不是也觉得[痛点]？"
- "为什么[现象]？原来是[原因]"
- "[数字]%的人不知道，[事实]"

### 冲突型
- "千万别再[错误做法]了！"
- "[常见认知]其实是错的"
- "我后悔没早点知道这个"

### 好奇型
- "这个[产品]到底有什么特别的？"
- "原来[意外发现]"
- "一条视频告诉你[答案]"

### 场景型
- "每次[场景]的时候，你是不是也..."
- "还记得上次[场景]吗？"

## 痛点/场景段模板（4-15秒）

### 食品类
```
[描述场景]，每次[痛点行为]
试过很多[同类产品]，但[不满意的地方]
直到我发现了[产品名]
```

### 美妆类
```
[肤质问题]困扰我很久了
用了好多[同类产品]都没效果
朋友推荐了[产品名]，[使用感受]
```

### 日用品类
```
[使用场景]的时候，最烦的就是[痛点]
市面上的[同类产品]要么[问题1]，要么[问题2]
[产品名]直接解决了这个问题
```

## 解决方案段模板（16-30秒）

### 产品展示型
```
你看[产品卖点1]
而且[产品卖点2]
现在下单还有[优惠信息]
点击下方链接/[行动号召]
```

### 效果证明型
```
用了[时长]，[效果描述]
[用户反馈/数据]
想要同款的[行动号召]
```

### 口碑推荐型
```
[产品名]真的绝了
[核心卖点]
我已经回购[次数]了
[行动号召]
```

## 完整示例

### 示例1：螺蛳粉带货

**分镜节奏：** 12帧Part1 + 12帧Part2

```
[Shot 01-02] 你有没有试过，半夜饿得不行，就想来点重口味的？
[Shot 03-06] 螺蛳粉我吃过不下二十个牌子，要么太臭，要么料少，要么粉不劲道。
[Shot 07-12] 直到我试了这款XX螺蛳粉，汤底是真正熬的，酸笋够味，花生酥脆，粉煮出来QQ弹弹的。
[Shot 13-16] 你看这个料包，满满当当的，每一样都是真材实料。
[Shot 17-20] 加班回家煮一包，十分钟搞定，比外卖还快。
[Shot 21-24] 现在买三送一，点击下方链接，今晚就能吃上！
```

### 示例2：护肤品带货

```
[Shot 01-02] 换季脸又开始起皮了？别再乱涂了。
[Shot 03-06] 我之前也是，面膜精华一大堆，钱花了不少，脸还是干。
[Shot 07-12] 后来皮肤科朋友让我试试这个XX精华，成分简单但真的有效。
[Shot 13-16] 你看这个质地，水润不粘腻，上脸就吸收了。
[Shot 17-20] 用了两周，起皮的地方全平了，摸起来滑滑的。
[Shot 21-24] 现在还有买二送一的活动，链接在下方，冲！
```

## 写作检查清单

- [ ] 前3秒有钩子吗？
- [ ] 每个 Part 的容量 QC 是否通过？
- [ ] 原片 must-keep / mergeable 功能是否都在台词或画面中有落点？
- [ ] 每句话能对应到具体镜头？
- [ ] 口语化，没有书面语？
- [ ] 有明确的行动号召（CTA）？
- [ ] 节奏跟得上画面切换？
