# 护肤 / 美妆 / 清洁泥膜复刻经验

用于泥膜、洁面、精华、面霜、面膜、染发、男士护肤等电商短视频复刻。目标是一次减少返工：人物像、产品稳、无字幕、口播清楚、前后效果成立。

## 核心原则

- 先听音频再写口播。字幕可能不全，必须从原视频或分段音频提取完整台词、停顿、语速和嘴型位置。
- 多段带声音时，先按 ASR 句子边界切 reference audio，再写 Seedance 提示词。不要机械切 15 秒导致上一段吃到下一段开头台词。
- 复刻动作含义，不要把某个动作拉满全段。护肤片常见结构是：问题近景 → 产品/质地 → 局部涂抹 → 擦掉/洗净 → 洗后证明。
- 复刻原片镜头和剧情，但产品形态必须按用户产品重映射。原片是泥膜棒、面膜袋或其他形态时，不要照搬旧道具；目标是白色圆罐泥膜时，就写开盖、取乳白泥膏、指腹/小勺/面膜刷局部涂抹。
- 产品名要适配原句时长。原片品牌句短时，新产品名不要硬塞全称；优先保证品牌名说完整。
- 画面内口播要和嘴型对应。正脸说话镜头必须开口，不能变成静止人脸配旁白。
- 默认无字幕。Seedance 很容易把逐句口播画成字幕，必须用无字幕提示词格式。
- 场景、画面功能或声音模式变化时必须新开执行块。画面内同期口播与画外旁白不能塞进同一块，短促静物 B-roll 单独成块。

## 视频分析必记

整理原片时，除了普通分镜，还要记录：

- 哪些秒数是同期声，哪些秒数是画外音。
- 哪些镜头嘴巴不开口，例如擦脸、涂抹、产品特写。
- 每句台词对应什么画面动作。
- 护肤前问题是什么：黑头、毛孔、油光、闭口、暗沉、粗糙。
- 护肤后证明是什么：更亮、更干净、更细腻、更清爽、不紧绷。
- 擦掉/洗净动作在哪里出现。毛巾擦脸、清水冲洗、纸巾擦拭这类镜头不能漏。

## 参考图准备

Seedance 稳妥输入顺序以 product profile 和 approved visual manifest 为准，泥膜常见为：

1. `图片1`：对应 Part 的最终改图分镜，只参考镜头顺序、动作和节奏。
2. `图片2`：紧裁产品参考图。产品主体必须占画面大部分，减少留白。
3. `图片3`：开盖/材质参考图，泥膜任务锁定乳白厚泥。
4. `图片4`：上半身人物身份图。保留脸、发型、衣服、肩线、袖口，不要只裁脸；不传递其背景，也不覆盖各 Shot 的皮肤状态。
5. `图片5`：仅当 profile 明确 `requires_afterwash_ref=true` 时使用的洗后脸部参考图。
6. `音频1`：对应 Part 的参考音频。

不要用过近人脸裁切当唯一身份图；它会丢衣服信息，导致短袖、长袖、背心乱跳。房间段要锁定上衣，健身房段或外景段另写允许换装。

产品原图如果留白很大，先做紧裁图再传 Seedance。白罐、绿叶、色块、瓶型这类识别点必须占比足够。产品文字很难逐字稳定时，写“文字无法稳定时保持简洁轻微虚焦，只保留瓶型、主色和标志图形”，不要让模型生成乱字。

只给当前 Part 和 product profile 需要的参考图。洗后改善默认由对应 Shot 的正向文案控制；不要为了“更稳”擅自增加洗后脸参考。产品特写段要在时间轴里再次 @ 产品图，开盖膏体段要 @ 开盖/膏体参考图。

## 产品动作映射

先提取原片动作功能，再映射到目标产品真实用法：

| 原片功能 | 目标是白色圆罐泥膜时的写法 |
|---|---|
| 拿着旧产品边说话 | 手拿 @白色圆罐 到脸旁口播 |
| 旧产品在脸上打圈 | 指腹/小勺取乳白泥膏，在鼻翼、脸颊、闭口区域局部推开 |
| 产品特写 | @白色圆罐 贴近镜头，罐身轻转，保留白罐和绿色叶形标 |
| 质地展示 | 打开 @开盖产品，露出乳白厚泥膏，用指腹或小勺轻触膏面 |
| 洗净证明 | 清水洗脸、毛巾擦干，再按 brief/profile 明确写出洗后状态；只有 profile 要求时才 @洗后脸部参考 |

提示词里只描述目标产品真实形态。不要写“不是泥膜棒”“不要生成棒状产品”这类污染词；直接写白色圆罐、开盖乳白泥膏、局部涂抹。

## 口播压缩规则

替换品牌/产品名时，按原句节奏控制字数：

- 原片短句：`就这个，白云山清洁泥膜`
- 可替换：`就这个，孔凤春清洁泥膜`
- 避免：`就这个，孔凤春马齿苋净颜清洁泥膜`

如果用户反馈品牌名没说完，只小改对应 Part 和对应句，不要重跑已合格 Part。把长产品名拆到后面卖点句里，或只在文案/详情页承载。

## 多段参考音频边界

护肤测评常在 15 秒附近跨一句话，不能直接把原音频按 `0-15s / 15-30s` 切开。标准做法：

- 先转写完整原片，记录每句起止时间。
- 每段参考音频只包含本段该说的完整句子；如果本段最后一句在 14.56 秒结束，就切到 14.56 秒，再补 0.44 秒静音或真实环境声到 15 秒。
- 下一段从 14.56 秒的新句开始，不从 15.00 秒硬切。
- Seedance 提示词最后 0.3-0.6 秒写成收手、合盖、擦脸、水声、环境声等动作余量，不能写“继续说话”。
- 生成前必须 ASR 转写每段 reference audio。检查 Part1 里没有 Part2 开头句，Part2 里没有 Part1 尾句。发现重叠先修音频，不用后期裁切补救。

示例：原片 `13.44-14.56 直接写在包装上了`，`14.56-16.12 而且它用起来就很方便`。Part1 应为 `0.00-14.56 原音频 + 0.44 秒静音`；Part2 从 `14.56` 开始。不能让 Part1 音频含有“而且它用”。

## Seedance 无字幕格式

如果 Seedance 生成画内字幕，不要继续用逐段 `口播："...台词..."` 的字幕脚本格式。改用这个结构：

```text
生成一条15秒、9:16竖屏真实护肤测评短视频。必须是无字幕版本：
NO CAPTIONS, NO SUBTITLES, NO BURNED-IN TEXT, NO ON-SCREEN WORDS.
画面里禁止出现任何口播字幕、中文字幕、标题条、说明文字、贴纸、水印、分镜边框、Shot编号。
下面出现的中文只用于音频和嘴型，不允许画成画面文字。

参考图绑定：
图片1是分镜节奏；图片2是紧裁产品；图片3是开盖/材质；图片4是上半身人物身份和服装。只有 profile 明确要求时才增加洗后脸部参考。

画面时间轴：
0.0-0.7秒，...
0.7-1.7秒，...

音频脚本只用于声音和嘴型，绝对不要显示成字幕：...
```

无字幕版仍要写嘴型要求：正脸说话时嘴巴自然开合；产品、膏体、毛巾擦脸镜头可以无明显口播嘴型。

官方无字幕提示写了仍出字幕时，不要通过裁切、遮挡或改变构图来规避。剧情和衔接优先，字幕后期再处理。

## 泥膜 / 清洁类动作模板

按原片实际顺序调整，不要机械套用：

- 0.0-1.0 秒：短促使用动作。不要让模型一直打圈或一直涂满全脸。
- 0.7-1.7 秒：毛巾/纸巾/清水擦掉或洗净。这个镜头如果原片有，必须明确写。
- 1.7-3.0 秒：黑头、毛孔、油光、闭口近景，展示问题。
- 3.0-6.0 秒：正脸口播 + 产品靠近镜头。
- 6.0-9.0 秒：开盖、膏体、局部涂抹。
- 9.0-12.0 秒：成分/质地/使用方式证明。
- 12.0-15.0 秒：若原片此处是洗后证明，保留脸部特写和对应说话方式，并按 brief/profile 写清结果强度。

如果开头原片是“很快擦掉”，提示词要写“短促动作后马上擦掉”，不要写“持续涂抹/搓开/打圈”。

## 审核清单

每次出片后用 contact sheet 快速看：

- request JSON 是否真的传入了本段需要的分镜、产品、人物和音频；不能只看提示词。
- 提示词是否干净：没有“不是泥膜棒”“不是孤立商品图”“同Part1”“后续继承”“静止余量”“焦点锁”等内部说明。
- reference audio ASR 是否和本 Part 台词一致，相邻 Part 没有重复句。
- 是否无字幕、无标题条、无画内说明文字。
- 房间段人物脸、发型、上衣是否稳定。
- 产品瓶型、主色、标志图形是否稳定；是否有大面积乱码。
- 使用前问题是否可见，但不要夸张成脏脸或医疗问题。
- 洗后是否达到 brief/profile 要求的明确改善；不能自动追加“保留真实毛孔/皮肤纹理”削弱用户要求的商业效果。
- 毛巾擦脸/洗净动作是否按原片出现。
- 最后一段是否画面内开口口播，而不是旁白。

## 成功标准

以 2026-07-16 `job-011` 孔凤春清洁泥膜 Part2 实测通过案例为当前护肤泥膜 Seedance 基准，完整提示词见 `examples/seedance-20-kongfengchun-part2-validated.txt`：

- 剧情：必须按当次原片 source rhythm，不套固定模板；清洗、洗后证明、手机打卡、空罐、B-roll 和产品收口等原片 must-keep beat 不得丢。
- 人物：同一张脸、同一发型、同一件上衣；头肩和上半身完整入画，不用裁切遮挡解决问题。
- 产品：白色圆罐和绿色叶形标稳定；开盖时是乳白厚泥膏；原片旧产品形态只复刻功能，不复刻道具形态。
- 洗后结果：同一人物身份不变；按 profile 达到干净透亮、平滑细腻，黑头、闭口、粉刺、油光和粗大毛孔消失，近距离呈现完全无瑕疵。
- 尾部硬切：静物 B-roll 独立成块；B-roll 前后的主播产品近景使用匹配的原片场景、景别和产品位置，不重新起手推近。
- 音频：按 ASR 句子边界切 reference audio；相邻 Part 没有重复句；最后尾段可加 0.3-1.0 秒容错让台词说完。
- 提示词：每段独立，使用 @图片 引用；产品特写和开盖镜头再次 @ 产品图；无内部纠错说明和污染词。
- 质检：保存 request JSON；抽 1fps contact sheet；跑 freezedetect；默认快速确认无静止、黑场、明显错人错产品错场景后交付。最终成片 ASR 只在用户要求核对台词/声音，或问题明确指向漏话、重复话、口播边界时运行。
- 实测证据：`output/job-011/experiments/part2_flawless_matchcut_retry_20260716/validation_report.md`。

## 只重跑失败部分

护肤片容易“一处修好，另一处变坏”。如果 Part2 已经合格，只重跑 Part1；如果只是品牌句太长或音频边界错，只改该句/该 Part 和对应 reference audio。保留 request JSON，确认素材顺序、asset 引用、音频 URL 没有变错。
