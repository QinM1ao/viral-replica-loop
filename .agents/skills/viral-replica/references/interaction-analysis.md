# Interaction Analysis

Use this for voiceover, seam, Seedance prompt, request QC, generation, and final QC.

## Voice And Rhythm

- Combine subtitle timing, ASR timing, and visual shots.
- Bind product name and selling points to the same shot function as the source video.
- Put spoken lines in Chinese quotation marks when writing Seedance prompt audio scripts.
- Do not fill all 15 seconds with speech; leave space for operation, product closeups, and seam motion.

## Seam

- Do not start the next part with a static face.
- The next part should begin in motion: hand movement, breath, blink, slight camera move, product action, or continuing operation.
- The previous part should end with an action or product state that the next part can inherit.

## Seedance Prompt

- Use clean prompt text only.
- Do not include old rejected details unless they still exist in approved references.
- Split product references when different images serve different roles.
- If a washed-skin reference is used, bind it only to after-wash proof shots.
- Stop before paid or batch generation.

## Final QC

- Run ffprobe and freeze detection.
- Run final ASR for sound-enabled videos.
- Check product name, seam, duplicated lines, rushed speech, face continuity, product continuity, and visible subtitles.
- Stop for subjective quality review.
